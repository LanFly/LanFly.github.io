API列表

下面所列出来的API除非特别声明，否则一般的调用方式都是：

LanEditor.ObjectName.functionName

function API

  1. LanEditor.GetRenderHTML()

该函数返回正在编辑的文件的渲染后的HTML文档。

Object API

  1. File

    1. ExportFile(fileName, content)

    下载文件。调用该函数将使浏览器发生下载事件。fileName是文件名，content是文件内容

    2. SaveFileToLocal(fileName, content)

    储存文件到localstorage。如果不存在该文件，则会新建文件，文件内容为content。如果已存在文件，则会以content覆盖原文件。可以使用该函数新建一个空文件，只要content传递一个空字符串。成功返回”OK”,失败则返回相应的字符串。其可能返回值为

    1. "LocalStorage not support" ：不支持localstorage

    2. "param wrong" ：参数为null

    3. DeleteFileFromLocal(fileName)

    从localstorage删除文件。删除成功返回true。不存在文件返回false。

    4. GetFileList()

    获取所有文件列表。返回一个对象数组。数组长度是文件个数。对象包含2个属性：

    1. name：文件名
    2. time：文件创建的时间

    5. GetFileContent(filename)

    获取文件内容。返回文件的内容。如果不存在文件返回false。

  2. Time

    1. GetTimestamp()

    返回当前UNIX时间戳

    2. GetTimeString(timestamp)

    将UNIX时间戳转换为字符串格式：2016/1/20 下午9:29:39

  3. Fn

    1. register(type, fn)

    注册事件。当事件发生时，将通知注册的函数。并按照注册先后顺序依次执行。

    第一个参数事件类型。第二个参数是对象。对象里面为要注册的函数。

  4. Keyword

    1. extend(keyword)

    扩展词库。扩展的所有词库将加入查找列表。

    参数是一个对象。内容为要扩展的词库数组。

TextElem API

该API需获取TextElem参数才能访问，TextElem为编辑区的jquery对象，在扩展函数中可以获取。

访问方式：

TextElem.functionName

下面列出所有方法

1. iGetFieldPos()

获取光标所在位置

2. iSelectField(start, end)

选中指定位置内字符或设置光标位置。
从start起选中(含第start个)，到第end结束（不含第end个）。
若不输入end值，即为设置光标的位置（第start字符后）。

3. iSelectStr(str)

选中指定字符串。str是要选中的字符串，如果不存在该字符串，则不选中。

4. iAddField(str)

在光标之后插入字符串。

5. iDelField(n)

删除光标前面(+)或者后面(-)的n个字符。如果n为负数，则删除光标后面的n个字符。

6. iGetPosStr(n)

获取光标前面或后面指定个数字符。n为负数则获取前面n个字符，正数获取后面n个字符。

7. iGetSpaceNum()

获取前一行的空格缩进个数或是本行前面的空格缩进个数。如果光标前一个字符不是换行\n或\r\n，则获取本行前面的缩进，否则获取上一行。
